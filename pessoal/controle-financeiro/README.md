# 📱 App de Controle Financeiro

Este é um projeto pessoal focado no design de interface (UI/UX) e na prototipagem de um aplicativo de controle de receitas e despesas pessoais. 

O projeto foi desenvolvido inteiramente no **Figma**, aplicando conceitos de usabilidade, arquitetura de informação e design responsivo.
